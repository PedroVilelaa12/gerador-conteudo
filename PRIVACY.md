# Privacy Policy

**Última atualização:** 2024

## 1. Coleta de Dados

Este aplicativo coleta apenas os dados necessários para funcionamento:

- **Credenciais de API:** Armazenadas localmente em arquivo `.env` no seu computador
- **Tokens de autenticação:** Obtidos das plataformas de redes sociais (TikTok, Instagram, LinkedIn)
- **Conteúdo gerado:** Armazenado localmente ou em serviços de armazenamento configurados (opcional)

## 2. Como Usamos os Dados

Os dados coletados são usados exclusivamente para:
- Autenticação com APIs de redes sociais
- Geração de conteúdo usando APIs de IA (Google Gemini)
- Publicação de conteúdo nas redes sociais autorizadas

## 3. Armazenamento

- **Local:** Todos os dados são armazenados localmente no seu computador
- **Arquivo .env:** Credenciais armazenadas em arquivo `.env` na raiz do projeto
- **Nenhum servidor externo:** Não enviamos dados para servidores externos (exceto APIs autorizadas)

## 4. Compartilhamento de Dados

**NÃO compartilhamos seus dados com terceiros**, exceto:
- Plataformas de redes sociais (TikTok, Instagram, LinkedIn) - para autenticação e publicação
- Google Gemini - para melhoria de prompts e geração de imagens (quando configurado)
- AWS S3 - para armazenamento de arquivos (opcional, quando configurado)

## 5. Segurança

- Credenciais são armazenadas em arquivo local (`.env`)
- **IMPORTANTE:** Não compartilhe o arquivo `.env` publicamente
- Mantenha o arquivo `.env` seguro e fora do controle de versão

## 6. Seus Direitos

Você tem o direito de:
- Acessar seus dados (armazenados localmente)
- Remover seus dados (deletando o arquivo `.env`)
- Revogar permissões das redes sociais a qualquer momento

## 7. Cookies e Rastreamento

Este aplicativo não usa cookies ou tecnologias de rastreamento.

## 8. Alterações na Política

Esta política pode ser atualizada periodicamente. Recomendamos revisar esta página regularmente.

## 9. Contato

Para questões sobre privacidade, entre em contato através do repositório do projeto.
